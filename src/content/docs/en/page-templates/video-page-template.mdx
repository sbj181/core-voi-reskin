## Video Modal/Popup/Lightbox

These button components are set up to accept the props 'text' to control the button's text, and 'videoLink' to control what video opens in the modal.  Videos are set via these buttons, making it easy to assign when writing content.  The videos are set up to stop/deactivate when you close the modal.
